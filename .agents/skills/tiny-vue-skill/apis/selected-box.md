## selected-box

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| config | Object |  | 已选栏配置项，详细属性请参考基本示例 |
| inverse | Boolean |  | 启用反选，属性 inverse 设置为 true，属性 total 设置为总记录数，启用反选模式 |
| select | Array |  | 已选栏数据列表 |
| total | Number |  | 已选择数据总条数 |

### Events

| 事件名 | 回调参数 | 说明 |
|--------|----------|------|
| change |  | 增加、删除、清空和拖拽顺序，都会触发组件 change 事件，事件参数就是组件当前暂存的数据 |
| clear |  | 清空按钮点击事件 |
| delete |  | 删除按钮点击事件 |
| drag |  | 拖动排序事件 |

### Slots

| 插槽名 | 说明 |
|--------|------|
| button | 已选栏右侧自定义按钮插槽 |
| clear | 清空按钮插槽 |
| close | 删除按钮插槽 |
| option | 已选择数据列表显示插槽 |
| select | 已选中数据条数显示插槽 |
