## divider

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| border-style | string | 'solid' | 设置分隔线的样式，该属性的可选值为 solid / dashed / dotted |
| color | string |  | 设置分隔线的颜色 |
| content-background-color | string |  | 设置分割线文案的背景颜色 |
| content-color | string |  | 设置分隔线文案的颜色 |
| content-position | string | 'center' | 设置分隔线文案的位置，该属性的可选值为 left / center / right |
| direction | string | 'horizontal' | 设置分隔线的方向，该属性的可选值为 horizontal / vertical |
| margin | string |  | 设置分割线上下左右的边距 |
| offset | string |  | 设置文本位置偏移量，需同时配合`text-position`属性使用，以确定偏移的方向 |
| height | string |  | 设置整体的高度 |
| font-size | string | 14px | 文本字体大小 |
| status | string |  | 分割线状态 |

### Slots

| 插槽名 | 说明 |
|--------|------|
| default | 默认插槽 |
