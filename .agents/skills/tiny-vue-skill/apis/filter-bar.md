## FilterBar

### Props

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| data | Array | [] | 过滤栏列表数据。 |
| v-model | Array | [] | 绑定值 |

### Events

| 事件名 | 回调参数 | 说明 |
|--------|----------|------|
| click | Function() | 过滤项点击事件。 |

### Methods

| 方法名 | 返回值 | 说明 |
|--------|--------|------|
| click(index) |  | 过滤项点击方法，index 是当前项索引 |

### Slots

| 插槽名 | 说明 |
|--------|------|
| default | 过滤项插槽。 |
| icon | 组件右侧图标插槽。 |
