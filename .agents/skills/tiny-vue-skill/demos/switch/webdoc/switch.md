# switch Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | <p>通过 <code>v-model</code> 绑定开关值。</p> | switch/basic-usage.vue |
| mini-mode | 迷你尺寸 | <p>通过 <code>mini</code> 设置小尺寸开关。</p> | switch/mini-mode.vue |
| width | 自定义宽度 | <p>通过 <code>width</code> 设置开关的宽度。</p> | switch/width.vue |
| loading | 加载中状态 | <p>通过 <code>loading</code> 设置开关显示加载状态。</p> | switch/loading.vue |
| custom-open-close | 自定义开关显示 | <p>通过 <code>show-text</code> 配置是否显示开关文本，开启配置后，可以使用具名插槽 <code>open</code> 和 <code>close</code> 自定义开关内容。</p> | switch/custom-open-close.vue |
| before-change | 变更前置处理 | <p>通过 <code>before-change</code> 设置开关变化前置处理逻辑，参数为一个回调函数，调用后完成变更。</p> | switch/before-change.vue |
| custom-true-false-value | 自定义开关取值 | <p>通过 <code>true-label</code> 设置开关开启时的值， <code>false-label</code> 设置关闭时的值。</p> | switch/custom-true-false-value.vue |
| dynamic-disable | 禁用状态 | <p>通过 <code>disabled</code> 设置开关禁用状态。</p> | switch/dynamic-disable.vue |
| custom-open-close-icon | 自定义开关图标 | <p>通过具名插槽 <code>active-icon</code> 和 <code>inactive-icon</code> 自定义开关图标。</p> | switch/custom-open-close-icon.vue |
| event-change | 变更事件 | <p>当开关值变化后，会触发 <code>change</code> 事件。</p> | switch/event-change.vue |
