| expand-control | 节点的展开控制 | <br>          通过 <code> default-expand-all </code> 属性，加载时默认展开所有节点，默认为 <code>false</code>。<br><br>          通过 <code> default-expanded-keys </code> 属性，加载时默认展开指定数组里的节点，需要指定<code>node-key</code>的值。<br><br>          通过 <code> expand-on-click-node </code> 属性，设置节点在点击内容时展开 / 收起，默认为 <code>true</code>。当设置为 <code>false</code>时，只有点击图标才展开。<br><br>          通过 <code> expandAllNodes </code> 组件方法，可以全部展开或收起所有节点。函数入参为 <code>true</code>时全部展开，反之则全部收起。<br><br>          通过 <code> node-expand </code> 事件，监听节点展开的动作。<br><br>          通过 <code> node-collapse </code> 事件，监听节点收起的动作。<br><br>          <div class="tip custom-block"><br>            <p class="custom-block-title"> 实用技巧 </p><br>            节点对象上包含<code>expand</code> 和 <code>collapse</code> 方法，可以展开或收起该节点。<br><br>          </div><br>           | tree/expand-control.vue |
