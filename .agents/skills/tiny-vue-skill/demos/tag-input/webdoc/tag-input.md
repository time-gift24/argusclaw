# tag-input Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | 按enter回车键添加标签，按backspace删除最后一个标签。<br> | tag-input/basic-usage.vue |
| disabled-readonly | 禁用与只读 | 你可以设置TagInput被禁用或者只读。<br> | tag-input/disabled-readonly.vue |
| max-tag | 最大标签数 | 您可以设置添加标签的数量限制。<br> | tag-input/max.vue |
| collapsed-tag | 折叠标签 | 通过设置minCollapsedTags属性，可以控制折叠标签的数量，超过部分将以+N的形式显示。<br> | tag-input/collapsed-tag.vue |
| clearable-tag | 可清空标签 | 通过设置clearable属性，可以控制标签是否可清空。<br> | tag-input/clearable-tag.vue |
| separator-tag | 分隔符输入标签 | 可以通过设置分隔符separator来实现批量输入。<br> | tag-input/separator-tag.vue |
| prefix-suffix | 自定义前后缀 | 可以通过设置prefix和suffix属性来自定义前后缀。<br> | tag-input/prefix-suffix.vue |
| draggable-tag | 可拖拽标签 | 可以通过设置drag属性来实现标签的拖拽功能。<br> | tag-input/draggable-tag.vue |
