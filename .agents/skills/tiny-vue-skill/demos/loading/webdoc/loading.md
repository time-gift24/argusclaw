# loading Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | <p>通过静态方法 <code>Loading.service</code> 在指定 <code>target</code> 上设置加载。</p><br> | loading/basic-usage.vue |
| size | 尺寸 | <p>可通过 <code>size</code> 属性设置尺寸大小，可选值：<code>medium</code> / <code>small</code> / <code>large </code>。</p><br> | loading/size.vue |
| loading-tip-text | 加载提示文本 | <p>通过 <code>text</code> 自定义加载文字的提示文本。</p><br> | loading/loading-tip-text.vue |
| background | 遮罩背景色 | <p>通过 <code>background</code> 自定义背景颜色和透明度。</p><br> | loading/background.vue |
| custom-class | 样式 | <p>通过 <code>customClass</code> 指定类名进行样式修改。</p><br> | loading/custom-class.vue |
| directive | 指令 | <p>通过 v-loading 使用指令。</p> | loading/directive.vue |
| fullscreen | 全屏加载 | <p><div>通过 <code>v-loading.lock.fullscreen</code> 指令添加修饰符进行全屏加载。</div><div>也能通过<code>Loading.service</code>并配置属性进行全屏加载。</div></p> | loading/fullscreen.vue |
| spinner | 加载图标 | <p>通过 <code>spinner</code> 自定义加载图标。</p><br> | loading/spinner.vue |
