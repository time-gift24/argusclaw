# button Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | 通过 <code>type</code> 设置按钮类型， <code>plain</code> 设置是否为朴素按钮，<code>round</code> 设置是否为圆角按钮， <code>circle</code> 设置是否为圆形按钮。 | button/basic-usage.vue |
| ghost | 幽灵按钮 | <p>通过 <code>ghost</code> 设置幽灵按钮。</p> | button/ghost.vue |
| dynamic-disabled | 动态禁用按钮 | <p>通过 <code>disabled</code> 设置是否禁用按钮。</p> | button/dynamic-disabled.vue |
| icon | 图标按钮 | 通过 <code>icon</code> 属性，设置按钮展示图标。<code>custom-style</code>属性设置按钮样式。 | button/icon.vue |
| text | 文字按钮 | <p>将 <code>type</code> 属性设置为 <code>text</code> 后展示形式为文字按钮，可通过 <code>text</code> 设置按钮文本。</p> | button/text.vue |
| image | 图片按钮 | <p>通过默认插槽自定义按钮内容，例如引入图片。</p> | button/image.vue |
| loading | 显示加载中 | <p>通过 <code>loading</code> 设置按钮是否为加载中的样式。</p> | button/loading.vue |
| size | 尺寸 | <p>通过 <code>size</code> 设置按钮尺寸，可选值为 <code>large</code> 、<code>medium</code> 、<code>small</code> 、<code>mini</code> 。不设置时为默认尺寸。</p> | button/size.vue |
| reset-time | 防止重复点击 | <p>通过 <code>reset-time</code> 设置单击后按钮禁用的时长，默认禁用时长为 1000 毫秒，可用于防止按钮连续点击出现表单重复提交的问题。</p> | button/reset-time.vue |
| autofocus | 默认聚焦 | <p>通过 <code>autofocus</code> 设置按钮原生属性，是否默认为聚焦状态。</p> | button/autofocus.vue |
| click | 点击事件 | <p>当点击按钮后会触发 <code>click</code> 事件。</p> | button/click.vue |
