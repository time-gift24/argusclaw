# space Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-space | 基本用法 | <p>默认采用横向布局（row），自动为插槽内容添加间距。</p> | space/basic-usage.vue |
| space-size | 间距尺寸 | <p>通过 <code>size</code> 属性设置间距大小，支持 small、medium、large 预定义值或自定义数值/数组。</p> | space/space-size.vue |
| space-direction | 排列方向 | <p>通过 <code>direction</code> 属性设置布局方向，支持 row（横向）或 column（纵向）。</p> | space/space-direction.vue |
| space-wrap | 换行显示 | <p>通过 <code>wrap</code> 属性控制子项内容是否换行显示。</p> | space/space-wrap.vue |
| space-align | 交叉轴对齐 | <p>通过 <code>align</code> 属性设置交叉轴对齐方式，支持 start、center、end、baseline 等值。</p> | space/space-align.vue |
| space-justify | 主轴对齐 | <p>通过 <code>justify</code> 属性设置主轴对齐方式，支持 start、center、end、space-between、space-around、space-evenly。</p> | space/space-justify.vue |
| space-order | 自定义排序 | <p>通过 <code>order</code> 属性传入 key 数组来自定义子元素的渲染顺序，未设置 key 的子元素将自动排列在最后。</p> | space/space-order.vue |
