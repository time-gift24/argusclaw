# user-contact Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | 通过<code>:data</code>设置数据源。<code>imgUrl:图片路径</code><code>userName:姓名</code><code>userDescription:用户描述</code>。 | user-contact/basic-usage.vue |
| data-source | 设置数据源 | <p>通过 <code>:data</code> 设置数据源，<code>imgUrl</code> 表示的是图片路径，<code>userName</code> 表示的是姓名，<code>userDescription</code> 表示的是用户描述。</p><br> | user-contact/data-source.vue |
| not-displayed-content | 定制弹框不显示的内容 | <p>通过设置<code>show-img</code> 是否显示头像，<code>show-name</code> 是否显示名字，<code>show-number</code> 是否显示用户账号，默认值为<code>true</code>。</p> | user-contact/not-displayed-content.vue |
| slots-usercontact | 自定义默认插槽 | 通过在组件内插入<code>div</code>元素即可。 | user-contact/slots-usercontact.vue |
| contact-espace | 连接 eSpace | 通过设置<code>espace</code>属性传值即可。 | user-contact/contact-espace.vue |
