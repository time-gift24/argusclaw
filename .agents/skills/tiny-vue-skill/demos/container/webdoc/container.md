# container Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基础用法 | <br>          组件接受<code>header</code>、<code> aside </code>、<code> footer </code>、<code> default </code> 四种插槽。<br><br>          通过<code> pattern </code> 来设置预定好的组件版型，版型决定了插槽是否显示以及显示位置。<br><br>         | container/basic-usage.vue |
| custom-with-height | 自定义宽度和高度 | <br>          通过 <code>headerHeight</code> 控制 header（头部区域）的高度。<br><br>          通过 <code>footerHeight</code> 控制 footer（底部区域）的高度。<br><br>          通过 <code>asideWidth</code> 控制 aside（左侧区域）的宽度。<br><br>          而 main（主体区域）的宽度和高度是自适应的。<br>         | container/custom-with-height.vue |
