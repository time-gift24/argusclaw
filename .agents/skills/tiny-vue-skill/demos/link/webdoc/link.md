# link Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | <br>          通过 <code>value</code> 或者默认插槽设置链接显示内容。<br><br>          绑定 <code>click</code> 事件监听点击。当禁用或配置 <code>href</code> 属性时，不会触发<code>click</code>事件！<br>         | link/basic-usage.vue |
| link-style | 主题样式 | <p>通过 <code>type</code> 设置链接的主题类型。</p> | link/link-style.vue |
| dynamic-disable | 禁用链接 | <p>通过 <code>disabled</code> 设置是否禁用链接。</p> | link/dynamic-disable.vue |
| focus-no-underline | 聚焦时不显示下划线 | <p>通过 <code>underline</code> 设置鼠标悬浮是否显示下划线。</p> | link/focus-no-underline.vue |
| custom-icon | 自定义图标 | <p>通过属性 <code>icon</code>、默认插槽、<code>icon</code> 插槽都可以自定义图标。</p> | link/custom-icon.vue |
| config-href | 链接地址 | <p>通过 <code>href</code> 设置跳转连接，可配合原生属性 <code>target</code> 一起使用。</p> | link/config-href.vue |
| target | 打开方式 | <p>通过 <code>target</code> 配置链接打开方式，可选值有 <code>_self</code>、<code>_blank</code>、<code>_parent</code>、<code>_top</code>。</p> | link/target.vue |
