# popconfirm Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | <p>通过 <code>title</code> 设置标题内容， <code>message</code> 设置正文内容，通过 <code>reference</code> 插槽自定义触发显示的元素。</p> | popconfirm/basic-usage.vue |
| trigger | 触发方式 | <p>通过 <code>trigger</code> 设置触发显示气泡确认框的方式。</p> | popconfirm/trigger.vue |
| type | 图标类型 | <p>通过 <code>type</code> 设置提示的图标类型，或者传入自定义图标。</p> | popconfirm/type.vue |
| placement | 提示框显示位置 | <p>通过 <code>placement</code> 设置提示框显示位置。</p> | popconfirm/placement.vue |
| custom-class | 自定义类名 | <p>通过 <code>custom-class</code> 自定义气泡提示框的类名。</p> | popconfirm/custom-class.vue |
| slot-footer | 底部插槽 | <p>通过 <code>footer</code> 插槽自定义底部内容。</p> | popconfirm/slot-footer.vue |
| slot-message | 消息插槽 | <p>通过 <code>message</code> 插槽自定义消息内容。</p> | popconfirm/slot-message.vue |
| width | 宽度 | <p>通过 <code>width</code> 设置提示框宽度。</p> | popconfirm/width.vue |
| cancel-button | 隐藏取消按钮 | <p>通过 <code>cancel-button</code> 设置是否展示取消按钮。</p> | popconfirm/cancel-button.vue |
| event | 事件 | <p>当气泡确认框显示时触发 <code>show</code> 事件，当确认框隐藏时触发 <code>hide</code> 事件。<br />当点击底部确认按钮时会同时触发 <code>hide</code> 和 <code>confirm</code> 事件。</p> | popconfirm/event.vue |
