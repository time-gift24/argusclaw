# anchor Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | 通过 <code>links</code> 设置导航数据， <code>type</code> 设置类型。 | anchor/basic-usage.vue |
| is-affix | 固定模式 | 通过 <code>is-affix</code> 设置固定定位，不随页面滚动。 | anchor/is-affix.vue |
| set-container | 滚动容器 | 通过 <code>container-id</code> 设置滚动容器， <code>link-click</code> 监听锚点点击事件，阻止浏览器默认行为，实现单页面 <code>hash</code> 路由模式的页内跳转，在指定容器内滚动。<br/><br>            <code>mark-class</code> 设置自定义类名，高亮显示导航的目标元素。 | anchor/set-container.vue |
| offset-top | 顶部偏移量 | 通过 <code>offset-top</code> 设置锚点距离顶部的偏移量。 | anchor/offset-top.vue |
| change | change 事件 | 通过 <code>change</code> 监听锚点改变的事件。 | anchor/change.vue |
