| tree-table-tree-grid-index-checkfield | 选中关联字段 | <p>通过 <code>selectConfig</code> 属性中配置 <code>checkField</code> 为行字段名称，可以将行的选中与该字段关联。被关联字段应该为布尔取值，并且在父子关联时父行初始设置为选中，应保证其所有子行也设置为选中。</p> | grid/tree-table/tree-grid-index-checkfield.vue |
| tree-table-tree-grid-fixed-column | 冻结列 | <p>在列的标签上可以通过配置 <code>fixed=&quot;left&quot</code>; 属性把该列固定在左侧，配置 <code>fixed=&quot;right&quot</code>; 属性把该列固定在右侧。</p> | grid/tree-table/tree-grid-fixed-column.vue |
| tree-table-has-tree-expand | 检查树节点是否已展开 | <p>通过 <code>hasTreeExpand(row)</code> 方法可以检查树节点是否已展开，参数 <code>row</code> 为行数据对象。</p><br> | grid/tree-table/has-tree-expand.vue |

| tree-table-tree-grid-index | 树表展开序号列配置 | <p>在 <code>treeConfig</code> 属性里配置 <code>ordered</code> 为 <code>false</code> 展开行，序号列按父子级的结构排序。默认 <code>ordered</code> 属性为 <code>true</code> 展开行后序号列按序号排序。</p><br> | grid/tree-table/tree-grid-index.vue |
| tree-table-tree-grid-insert-delete-update | 增删改 | <p>在 <code>grid-toolbar</code> 标签上配置 <code>slot=&quot;toolbar&quot</code>; 插槽，配置相关的操作按钮，注意：新增数据需要在 <code>grid</code> 标签上配置 <code>edit-config</code> 编辑框的配置信息，并通过 <code>ref</code> 获取到组件实例调用方法进行相关操作，<code>findTree、filterTree</code> 是工具类里的查找和过滤方法。由于 tree</code> 数据节构的特殊性，<code>grid</code> 的 <code>getInsertRecords</code> 与 <code>getRecordset</code> 方法无法获取到新增数据，此类数据需要用户自行处理。</p><br> | grid/tree-table/tree-grid-insert-delete-update.vue |
