# grid Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| custom-column-width | 本地存储列宽 | <p>在 <code>grid</code> 标签上配置 <code>resizable=&quot;true&quot;</code> 可以调整列宽，<code>grid-column</code> 标签上配置 <code>resizable=&quot;false&quot;</code> 可以配置某一列不能调整列宽。并在 <code>toolbar</code> 标签上配置 <code>resizable=&quot;{storage: true}&quot;</code> 保存在本地的 <code>localStorage</code> 里，刷新页面表格的列宽按照拖动后的列宽显示。</p><br> | grid/custom/column-width.vue |
| custom-default-csutoms | 初始化个性配置 | <p>在 <code>grid</code> 标签上配置 <code>customs</code> 可以设置表格的初始化个性配置，可以控制表格列的隐藏，列宽等。</p><br> | grid/custom/default-customs.vue |
| custom-column-simple | 简化版列设置 | <p>在 <code>toolbar</code> 标签上配置 <code>:setting=&quot;{simple: true, customDisable: customDisable}&quot;</code> 开启简化版列设置 通过<code>customDisable</code> 方法自定义当前选项是否禁用。</p><br> | grid/custom/column-simple.vue |
| custom-reset-resizable | 手动重置列操作 | <div class="tip custom-block"><p class="custom-block-title">方法说明</p><br><p> <code>resetResizable()</code>：手动重置列宽拖动的操作，还原到初始状态<br><code>resetCustoms()</code>：手动重置列的显示/隐藏操作，还原到初始状态<br><code>resetAll()</code>：手动重置列的所有操作，还原到初始状态</p><br></div><br><p>请先对列宽进行拖动、隐藏列等操作，然后单击对应按钮查看效果。</p><br> | grid/custom/reset-resizable.vue |
| custom-column-sort | 排序 | <p>在 <code>column</code> 标签上配置 <code>sortable</code> 可以排序（未配置的列，在个性化面板中的排序图标会隐藏），并在 <code>toolbar</code> 标签上配置 <code>setting</code> 开启个性化设置，在个性化面板里进行排序，排序过后会保存在本地的 <code>localStorage</code> 里，刷新页面页面上会显示排序过后的数据。</p><br> | grid/custom/column-sort.vue |
| custom-multiple-column-sort | 多列排序 | <p> <code>表头排序</code> 只支持单列排序，<code>个性化排序</code> 可以通过配置 <code>sortConfig.multipleColumnSort</code> 开启多列排序，<br>          <code>sortConfig.multipleColumnSort</code> 支持配置为 <code>true</code> 或者 <code>方法</code>，配置为方法时可以自定义多列排序。</p><br> | grid/custom/multiple-column-sort.vue |
| custom-sort-type | 排序类型 | <p>在 <code>column</code> 标签上配置 <code>sortable</code> 可以排序，并在 <code>toolbar</code> 标签上配置 <code>setting</code> 开启个性化设置，在个性化面板里选择其它设置，其它设置里可以选择排序类型，排序类型有当前页排序和所有数据排序选项，默认是当前页排序。</p><br> | grid/custom/sort-type.vue |
| custom-column-fixed | 列冻结 | <p>在 <code>column</code> 标签上配置 <code>fixed=&quot;left&quot;</code> 可以固定在左侧，并在 <code>toolbar</code> 标签上配置 <code>setting</code> 开启个性化设置，在个性化面板里可以设置是否固定列。</p><br> | grid/custom/column-fixed.vue |
| custom-ordercolumn-remote | 加载远端数据 | <p>通过 <code>reloadCustoms</code> 方法，将第二个参数置为 <code>true</code> 可按序加载列。</p><br> | grid/custom/ordercolumn-remote.vue |
| custom-column-visible-hidden | 列显示隐藏 | <p>在 <code>toolbar</code> 标签上配置 <code>setting</code> 开启个性化设置，在个性化面板里可以设置是否显示和隐藏列。</p><br> | grid/custom/column-visible-hidden.vue |
| custom-hide-column | 手动隐藏显示列 | <p>通过调用 <code>hideColumn(column)</code> 方法隐藏指定列，<code>showColumn(column)</code> 方法显示指定列。使用这两个方法隐藏/显示列时，还需要再调用 <code>refreshColumn()</code> 方法刷新列配置才能实际生效。</p><br> | grid/custom/hide-column.vue |
| custom-page-size | 分页条数 | <p>在 <code>toolbar</code> 标签上配置 <code>setting</code> 开启个性化设置，在个性化面板里选择其它设置，其它设置里可以选择每页显示的条数，可以通过 <code>setting</code> 的 <code>other</code> 属性控制显示其他设置，<code>boolean</code> 类型默认为 <code>true</code>。</p><br> | grid/custom/page-size.vue |
| custom-server-storage | 服务端存储 | <br>        <p>在 <code>toolbar</code> 标签上配置 <code>:setting=&quot;{storage:'remote'}&quot;</code> 开启个性化并启用远程服务获取个性化数据，并配置 <code>@remote-setting</code> 调用 <code>reloadCustoms</code> 方法更新个性化数据。<br>         注意：如果是根据服务端数据控制显示/隐藏列，调用接口获取到个性化的数据后需手动调用 <code>reloadCustoms</code> 方法更新。</p> | grid/custom/server-storage.vue |
| custom-reset-method | 自定义重置 | <p>通过工具栏属性 <code>reset-method</code>  配置一个方法，在重置个性化时调用，需要返回 <code>promise</code>，指定重置的个性化数据，参考示例： | grid/custom/reset-method.vue |
| custom-ordercolumn-local | 个性化按钮点击事件 | <p>配置 <code>setting</code> 的 <code>storage</code> 为 <code>local</code> 可实个性化的本地存储，<br>          配置 <code>setting</code> 的 <code>sortable</code> 为 <code>sortablejs</code> 可实现拖拽排序列功能，此功能在多级列头下无效，<br>          并且可以通过三个事件<code>save-setting</code>、<code>cancel-setting</code>、<code>reset-setting</code>来分别监听确认、取消、重置按钮的点击事件。</p><br> | grid/custom/ordercolumn-local.vue |
| custom-prsonalized-drag | 个性化面板拖拽前事件 | <p>在 <code>toolbar</code> 标签上配置 <code>setting</code> 开启个性化设置，在个性化面板里配置 <code>onBeforeMove</code> 拖拽前事件，也可配置 <code>filter</code> 设置类名，拖拽个性化面板某一行会触发个性化面板的拖拽前事件。</p><br> | grid/custom/prsonalized-drag.vue |
