# grid Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| renderer-inner-renderer | 内置渲染器 | <br>        <p>内置渲染器设置当前表格列的显示获取类型。<br>在 <code>grid-column</code> 列配置 <code>format-text</code> 对象，配合 <code>format-config</code> 可以控制数据格式。</p><br>        <div class="tip custom-block"><br>          <p class="custom-block-title">TIP</p><br><p> <code>format-text</code> 的可取值包括：</p><br>          <ul><br>            <li><code>integer</code>：显示成整数。</li><br>            <li><code>number</code>：显示成小数。</li><br>            <li><code>money</code>：显示成钱数。</li><br>            <li><code>rate</code>：显示成百分比的数据。</li><br>            <li><code>filesize</code>：显示成文件大小的数据。</li><br>            <li><code>boole</code>：显示成对号或者叉号的图标。</li><br>            <li><code>date</code>：显示成年月日。</li><br>            <li><code>dateTime</code>：显示成年月日时分。</li><br>            <li><code>longDateTime</code>：显示成年月日时分秒。</li><br>            <li><code>time</code>：显示成时分。</li><br>            <li><code>longTime</code>：显示成时分秒。</li><br>            <li><code>yearMonth</code>：显示成年月。</li><br>            <li><code>enum</code>：通过数据的值，查询出要显示的文本。</li><br>            <li><code>ellipsis</code>：显示成省略的内容，鼠标移到该单元格上，显示完整内容的提示信息。需要同时配置 format 的 len 属性。</li><br>          </ul><br>        </div><br>         | grid/renderer/inner-renderer.vue |
| renderer-custom-renderer | 自定义渲染器 | <p>自定义渲染器可以自定义渲染单元格，也可渲染一个 <code>vue 组件</code>。在 <code>grid-column</code> 列配置 <code>renderer</code>，支持方法和对象配置。具体参考下面示例。</p><br> | grid/renderer/custom-renderer.vue |
| renderer-inner-renderer-date | 日期渲染器 | <p>在日期字段为字符串值时，需要给日期渲染器提供 <code>valueFormat</code> 配置才能正常解析日期字符串。</p><br> | grid/renderer/inner-renderer-date.vue |
| renderer-inner-renderer-rate | 比率渲染器 | <p>通过 <code>formatConfig.max</code> 配置，按照 <code>value / max</code> 计算样式百分比（进度条宽度占单元格宽度的百分比），按照 <code>value</code> 计算显示百分比（进度条上显示的比率），参考示例字段内置列渲染 <code>rate1</code>。<br>          如果未配置 <code>formatConfig.max</code>，就按照 <code>value / total</code> 计算样式百分比和显示百分比，<code>formatConfig.total</code> 默认值是 <code>1</code>，参考示例字段内置列渲染 <code>rate2</code>。</p> | grid/renderer/inner-renderer-rate.vue |
| render-async-colunm-render | 列异步数据渲染 | <p>将列<code>format-config</code>的 <code>async</code>属性配置为<code>true</code>后，支持动态修改其<code>data</code>数据自动渲染。这种整列异步渲染一般将<code>type</code>设置为<code>enum</code>，通过<code>value</code>与<code>label</code>可以配置渲染单元格的映射。</p><br> | grid/renderer/async-colunm-render.vue |
