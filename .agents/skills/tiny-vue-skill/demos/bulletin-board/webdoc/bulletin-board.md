# bulletin-board Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| base | 基本用法 | <p>通过 <code>data</code> 设置公告牌公告信息数据，<code>tab-title</code> 属性设置每个公告牌标题。</p><br> | bulletin-board/base.vue |
| title | 公告牌标题 | <p>通过 <code>title</code> 设置公告牌标题。</p><br> | bulletin-board/title.vue |
| active-name | 默认激活的选项卡 | <p>通过 <code>active-name</code> 设置默认激活的选项卡，从 <code>1</code> 开始计算。</p><br> | bulletin-board/active-name.vue |
| tab-title | 选项卡标题 | <p>通过 <code>tab-title</code> 设置自定义选项卡标题。</p><br> | bulletin-board/tab-title.vue |
| url | 基于 Url 的跳转 | <p>基于 <code>Url</code> 的跳转是通过数据中的 <code>url</code> 和 <code>target</code> 字段进行设置的。</p><br> | bulletin-board/url.vue |
| route | 基于 Route 的跳转 | <p>基于  <code>Route</code> 的跳转是通过数据中的 <code>route</code> 字段进行设置的的。</p><br> | bulletin-board/route.vue |
| more-link | 显示“更多”链接 | <p>通过 <code>more-link</code> 显示并设置更多链接。 <code>show-more</code> 设置是否显示更多按钮，默认为 true。</p><br> | bulletin-board/more-link.vue |
| icon | 自定义新公告前缀 | <p>通过 <code>icon</code> 自定义新公告前缀图标。</p><br> | bulletin-board/icon.vue |
| events | 事件 | <p>当点击内容后会触发 <code>contentClick</code> 事件。</p> | bulletin-board/events.vue |
