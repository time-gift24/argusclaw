# message Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | 通过<code>status</code>属性设置组件状态，可选值有<code>info</code>、<code>success</code>、<code>warning</code>、<code>error</code>、<code>loading</code>五种状态。<code>message</code> 属性设置提示信息。 | message/basic-usage.vue |
| message-duration | 自定义时长 | <code>duration</code> 属性用于设置消息提示的显示时长，单位为毫秒。 | message/message-duration.vue |
| message-close | 消息显示关闭按钮 | <code>message-closable</code> 属性用于设置消息提示是否可关闭。 | message/message-close.vue |
| prevent-duplicate | 防止重复消息提示 | <code>id</code> 属性设置防止重复提示。 | message/prevent-duplicate.vue |
