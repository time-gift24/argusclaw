| split-mode | 分割方式 | <p>同个 mode 设置分割类型，可选值为 <code>horizontal</code>（横向）或 <code>vertical</code>（纵向）；默认值<code>horizontal</code>。</p><br> | split/split-mode.vue |
| movestart-event | movestart 事件 | <p>开始移动事件</p><br> | split/movestart-event.vue |

| moveend-event | moveend 事件 | <p>移动完成事件</p><br> | split/moveend-event.vue |
| event-click | 点击事件 | <p><code>left-top-click</code>事件点击左箭头收起时触发， <code>right-bottom-click</code>事件点击右箭头收起时触发</p><br> | split/event-click.vue |
| left-right-slot | 左右面板插槽 | 详细用法参考如下示例 | split/left-right-slot.vue |

| nested-use | 嵌套使用 | 详细用法参考如下示例 | split/nested-use.vue |
| disabled-drag | 禁用拖拽 | 通过 <code> disabled </code> 来禁用拖拽 | split/disabled-drag.vue |

| horizontal-collapse | 简易模式中，双向展开 | 通过 <code> collapse-left-top、collapse-right-bottom  </code> 来启用简易模式的双向展开按钮 | split/horizontal-collapse.vue |
| horizontal-collapse-left-top | 简易模式中，单向展开 | 通过 <code> collapse-left-top 或 collapse-right-bottom  </code> 来启用简易模式的单向展开按钮 | split/horizontal-collapse-left-top.vue |
| three-areas | 支持配置 3 个区块 | 通过 <code> three-areas </code> 来启用三个区块模式 | split/three-areas.vue |
