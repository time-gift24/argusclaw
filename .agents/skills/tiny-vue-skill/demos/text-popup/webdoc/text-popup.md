# text-popup Demos

| demoId | 名称 | 描述 | 代码文件 |
|--------|------|------|----------|
| basic-usage | 基本用法 | 详细用法参考如下示例 | text-popup/basic-usage.vue |
| placeholder | 提示占位文本 | <p><code>placeholder</code> 属性的值为 <code>string</code> 此属性会把值默认显示到输入框中。</p><br> | text-popup/placeholder.vue |
| readonly | 只读 | <p>通过 <code>readonly</code> 设置是否只读。</p><br> | text-popup/readonly.vue |
| separtor | 分隔符 | <p>通过 <code>separtor</code> 设置分隔符，在文本框中输入回车就是分隔符的位置。</p><br> | text-popup/separtor.vue |
| modelValue | 绑定值 | <p>通过 <code>modelValue</code> 或 <code>v-model</code> 设置绑定值。</p><br> | text-popup/modelValue.vue |
| clear-value | 清除 value 值 | 详细用法参考如下示例 | text-popup/clear-modelValue.vue |
| width | 宽度 | <p>通过 <code>width</code> 设置宽度。</p><br> | text-popup/width.vue |
